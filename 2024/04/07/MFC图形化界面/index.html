MFC 正向开发

逆向思路

  • 通过 ResourceHacker 可以获取控件的 ID 号、修改控件布局等

  • 通过 xspy 可以获取控件消息所对应的处理函数地址

  • 通过 GetDlgItem / GetDlgItemText / GetDlgItemInt / GetWindowText 系列 API 找到关键函数

找到关键函数后，即为常规逆向
